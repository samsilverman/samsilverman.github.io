--- title: NavStick excerpt: NavStick is a self-directed navigation tool for people with visual impairments. permalink: /projects/project/navstick/ tags: - Interaction - Accessibility - Unity - Simulation - C# - Python redirect_from: - /navstick.html share: false read_time: false category: Research --- I worked in the Computer-Enabled Abilities Laboratory (CEAL), researching ways to improve the state of the art navigation tools for people with visual impairments. This research has lead to the development of NavStick which is our novel solution to blind and visually-impaired navigation.

My Work

  1. Developing NavStick and its user interface.
  2. Developing a virtual grocery store and video game level to test NavStick in a user study. This is done in Unity and is connected to a back-end server for test administrators to control.
  3. Designing and administered user studies of NavStick on blind participants.
  4. Writing a paper on results to be submitted to CHI 2021.
More detail on this project is to come as the deadline for CHI 2021 approaches. The official project page is to be apart of the CEAL website which is still in construction.
